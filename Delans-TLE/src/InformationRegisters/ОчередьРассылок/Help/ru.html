Очередь рассылок

В очереди рассылок можно следить за процессом рассылки сообщений получателям.

Состояние задания на рассылку отображается в заголовке формы:

  • Задание не запланировано - отсутствуют задания для рассылки. Запланировать новое задание можно из формы документа "Массовая рассылка" по кнопук "Отправить рассылку".
  • Задание запланировано - задание запустится в ближайшее время
  • Задание выполняется - выполняется процесс рассылки сообщений

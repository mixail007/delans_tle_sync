Добавление торговых предложений в список публикуемых

Для добавления торгового предложения в список публикуемых необходимо нажать команду Добавить.

Отмена публикации

Для отмены публикации торгового предложения необходимо выбрать торговое предложение в списке и нажать команду Удалить. Статус публикации изменится на Отмена публикации. Это действие можно отменить, нажав еще раз команду Удалить.

Синхронизация

Доступно два варианта синхронизации публикуемых торговых предложений:

  • Вручную — для этого необходимо выполнить команду Синхронизировать.
  • Автоматически — для этого необходимо установить флаг Автоматически синхронизировать и настроить расписание регламентного задания Синхронизация торговых предложений с сервисом 1С:Бизнес-сеть. Регламентное задание необходимо запускать под пользователем с назначенными правами доступа к сервису 1С:Бизнес-сеть.

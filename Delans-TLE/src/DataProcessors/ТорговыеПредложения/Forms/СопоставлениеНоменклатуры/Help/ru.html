Сопоставление номенклатуры

В форме сопоставления номенклатуры с сервисом 1С:Бизнес-сеть производится сопоставление видов номенклатуры, реквизитов номенклатуры и значений реквизитов номенклатуры. Сопоставление производится для отображения информации в форме просмотра торгового предложения и возможности использования отборов при поиске торговых предложений.

Сопоставление номенклатуры с 1С:Бизнес-сеть происходит следующим образом:

Сопоставление видов номенклатуры с категориями 1С:Бизнес-сеть

Сопоставление групп номенклатуры с категориями 1С:Бизнес-сеть

Сопоставление элементов номенклатуры с категориями 1С:Бизнес-сеть

Сопоставление реквизитов вида номенклатуры с реквизитами 1С:Бизнес-сеть

Сопоставление значений реквизитов со значениями реквизитов 1С:Бизнес-сеть

Сопоставление видов номенклатуры с категориями 1С:Бизнес-сеть

Сопоставление видов номенклатуры необходимо для поиска товаров по категориям Сервиса 1С:Бизнес-сеть.

Выгрузка торговых предложений производится по сопоставленным видам номенклатуры производится в рубрики сервиса 1С:Бизнес-сеть. Остальные торговые предложения публикуются в служебной рубрике Разное.

Порядок сопоставления:

  • В строке с необходимым Видом номенклатуры двойным кликом в колонке Категория 1С:Бизнес-сеть открыть окно Выбор категории товаров 1С:Бизнес-сеть.
  • Найти в дереве нужную категорию.
  • Нажать команду Выбрать.

Сопоставление групп номенклатуры с категориями 1С:Бизнес-сеть

В режиме сопоставления по группам номенклатуры, категории рубрикаторв 1С:Бизнес-сеть сопоставляются с группами номенклатуры по иерархии. Возможно сопоставление только общих реквизитов номенклатуры.

Сопоставление по группе номенклатуры имеет приоритет перед сопоставленным видом номенклатуры.

Сопоставление элементов номенклатуры с категориями 1С:Бизнес-сеть

В режиме сопоставления по элементам номенклатуры, категории рубрикаторв 1С:Бизнес-сеть сопоставляются с элементами справочника номенклатура. Возможно сопоставление общих реквизитов номенклатуры и по дополнительным реквизитам.

Сопоставление по элементам номенклатуры имеет приоритет перед сопоставленной группе или виду номенклатуры.

Если сопоставление по номенлатуре не выполнено, используется сопоставление по группе номенклатуры, по виду номенклатуры.

Сопоставление реквизитов вида номенклатуры с реквизитами 1С:Бизнес-сеть

Список реквизитов содержит доступные для сопоставления реквизиты номенклатуры, дополнительные реквизиты номенклатуры, дополнительные реквизиты характеристик номенклатуры, общие реквизиты номенклатуры, общие реквизиты характеристик.

Порядок сопоставления:

  • В списке Видов номенклатуры выбрать необходимую строку, в которой произведено сопоставление с категорией 1С:Бизнес-сеть.
  • В правом списке формы заполнится список доступных реквизитов сервиса 1С:Бизнес-сеть и сопоставленных ранее реквизитов. Для сопоставленных реквизитов указывается информация о сопоставлении, например, Сопоставлено 5 из 10.
  • В нужной строке списка реквизитов в колонке Реквизит вида номенклатуры выбрать реквизит из выпадающего списка.

Сопоставление значений реквизитов со значениями реквизитов 1С:Бизнес-сеть

В зависимости от типов значений реквизитов информационной базы и значений реквизитов 1С:Бизнес-сети сопоставление значений может происходить автоматически или вручную.

Порядок сопоставления:

  • Автоматическое сопоставление происходит для типов Число и значения Да/Нет. При этом в поле Сопоставлено указывается режим сопоставления Автоматически.
  • Ручное сопоставление происходит для ссылочных типов. При этом в колонке Тип значения указывается Список, в колонке Сопоставлено указывается информация о сопоставлении, например Сопоставлено 5 из 10. Для сопоставления значений необходимо нажать на гиперссылку и открыть окно Сопоставление значений реквизита 1С:Бизнес-сеть, где производится сопоставление значений.

При ручном сопоставлении необходимо по гиперссылке открыть окно Сопоставление значений реквизита 1С:Бизнес-сеть.
Выбор значения производится в выпадающем списке значений в колонке Значение 1С:Бизнес-сеть. По команде Заполнить сервис подставляет совпадающие по наименованию значения.

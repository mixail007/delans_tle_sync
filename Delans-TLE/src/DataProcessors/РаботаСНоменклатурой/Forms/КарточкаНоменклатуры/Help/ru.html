Карточка номенклатуры

В форме отображается информация об основных реквизитах номенклатуры, сведения о производителе, описание.
Если из карточки уже была создана номенклатура, из формы просмотра по гиперссылке в поле Номенклатура программы можно перейти к элементу справочника номенклатура. Если в базе ее нет, будет отображаться гиперссылка Создать.

Для перехода к категории сервиса, в которой представлена номенклатура, есть возможность перейти по строке пути Категория.

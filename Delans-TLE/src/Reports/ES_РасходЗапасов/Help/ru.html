Отчет "Анализ доступности запасов"

Отчет позволяет проанализировать остаток запасов, резерв и свободное количество товара.

Отчет обладает вертикальными и горизонтальными группировками. По вертикали группируются структурные единицы предприятия. По горизонтали отчет группируется по организациям и номенклатурным позициям. Таким образом, с помощью отчета можно определить количество зарезервированного товара по всем структурным единицам суммарно и оценить резерв и свободный остаток по предприятию.

Факт резервирования и размещения регистрируется документами, сформированными с указанием документа Заказ покупателя. Существует возможность настроить отчет с отображением зарезервированного товара по каждой структурной единице.

Группы доступа контактов и контрагентов

Справочник предназначен для описания доступа к элементам справочников Контакты и Контрагенты  по виду доступа Группы контактов и контрагентов. Справочник используется в том случае, если необходимо ограничить доступ пользователям к конкретным контактам и/или контрагентам. Информация о том, к какой группе доступа принадлежит тот или иной контакт/контрагент, указывается в справочнике Контакты/Контрагенты. Предусмотрена возможность отнесения контактов/контрагентов к определенной группе доступа с помощью сервисного механизма группового изменения реквизитов, который можно вызвать для выделенных контактов/контрагентов из формы списка справочника Контакты/Контрагенты.

Пример. Менеджеры по продажам работают с различными контрагентами. Одни менеджеры работают с оптовыми покупателями, а другие с розничными. Менеджерам должна быть доступна информация только по тем контрагентам и связанным с ними контактам, с которыми они работают и не должна быть доступна информация по контрагентам и связанным с ними контактам других менеджеров. В справочник Группы доступа контактов и контрагентов вводится информация о всех тех группах контактов/контрагентов, с которыми работают менеджеры: Оптовые покупатели и Розничные покупатели. Для каждого пользователя в качестве разрешенной группы доступа контактов/контрагентов назначается соответствующая группа доступа: Оптовые покупатели или Розничные покупатели.

Таким образом, для менеджеров будет ограничен список доступных им контрагентов и так же будет ограничен список доступных контактов, связанных с данными контрагентами:

  • менеджеры по работе с оптовыми покупателями будут видеть информацию только по тем контрагентам/контактам, для которых определена группа доступа контактов и контрагентов Оптовые покупатели
  • менеджеры по работе с розничными покупателями будут видеть информацию только по тем контрагентам/контактам, для которых определена группа доступа контактов и контрагентов Розничные покупатели


Важно. Установить ограничения по группам доступа контактов и контрагентов возможно только в том случае, если в разделе Настройки/Настройки пользователей и прав установлены опции Ограничивать доступ на уровне записей и Группы доступа контактов и контрагентов.

Справочник "Виды заказов покупателей"

Справочник предназначен для группировки различных состояний заказов покупателей по бизнес-процессам.

Каждый вид заказа будет представлять собой отдельную воронку продаж.

Для ввода дополнительных видов требуется включение функциональной опции "Виды заказов покупателей".

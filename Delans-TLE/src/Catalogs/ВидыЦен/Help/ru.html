Справочник "Виды цен"

Справочник предназначен для хранения видов отпускных цен компании.

 

В разрезе видов отпускных цен строится прайс-лист, поэтому в программе виды цен еще называются колонками прайс-листов. Каждая позиция номенклатуры может иметь столько цен, сколько в системе зарегистрировано видов цен.

Отпускные цены используются для подстановки по умолчанию в документах отгрузки, в заказах покупателей. Один из видов отпускных цен в системе имеет специальное назначение. Это вид цен, который содержит данные не о ценах для покупателей, а содержит значение учетной цены номенклатуры. Учетная цена может использоваться, например, как плановая себестоимость номенклатуры.

Цена может указываться вручную, загружаться из внешнего источника (например, файл Excel), а может рассчитываться автоматически. Цены, которые зависят от других называются Динамическими. Динамические цены рассчитываются по базовым ценам (на процент) или по формуле. В случае, когда выключен флаг автоматического расчета динамических цен формирование новых записей происходит в ручном режиме.

Для цены может быть заполнен формат и правило округления.

Цены разных видов цен можно сравнивать между собой при помощи отчетов.

Валюта храниться в карточке вида цен и является общей для всех цен колонки прайс-листа, т.е. в рамках одного вида цен невозможно указать цены в разных валютах. Для динамических цен, которые рассчитываются процентом валюта наследуется из базового типа цен.

ВАЖНО! Для динамических видов цен (формула) программа отслеживает изменение курса только для тех валют, у которых включен признак "Вводить вручную". Для остальных динамических видов цен, у которых выбрана валюта с признаком отличающимся от "Вводиться вручную" пересчет по новому курсу валюты инициализируется самостоятельно из карточки элемента справочника Виды цен по команде "Регистрировать новые цены и закрыть"

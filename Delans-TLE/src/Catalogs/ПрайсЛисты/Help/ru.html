Справочник "Прайс-листы"

Справочник хранит настройки формирования прайс-листов. Настройки условно можно разделить на следующие группы:

 

  • Внешний вид;
  • Отборы номенклатуры;
  • Включаемые виды цен (колонки с ценами);
  • Дополнительная информация;

 

Внешний вид

- Вид прайс-листа. Предоставляется возможность выбора вида компоновки: полотно, две колонки, диафильм.

  • Полотно – является классическим вариантом компоновки данных, при котором номенклатура, ее описание, цены и другая информация располагаются в строку, а сам прайс-лист выглядит в форме одной монолитной таблицы.
  • Две колонки – представляет собой способ подачи информации с максимальной компактностью. В связи с этим имеет некоторые ограничения, например, выводится только одна колонка с ценой, отсутствует возможность вывода картинки и других второстепенных данных.
  • Диафильм (слайды) – по внешнему виду напоминает каталог товаров. Основное назначение – предоставить пользователю максимально наглядный прайс-лист, где номенклатура представлена в виде картинок.

- Иерархия. Опция позволяет выбрать, по какому справочнику построить дерево прайс-листа. Кроме классического варианта, с иерархией по папкам номенклатуры, присутствует возможность построить прайс-лист по элементам ценовых групп или категорий номенклатуры. Т.е. справочник номенклатуры может содержать один набор папок, а иерархия прайс-листа может отличаться. Примером использования, помимо прочих, могут быть случаи, когда папки номенклатуры заводились по производителям, а прайс-лист необходимо построить по продукции и фактически объединить всех производителей в одну группу прайс-листа.

- Валюта. Указывается валюта прайс-листа и дата для курса, по которому будут пересчитываться цены.

 

Отборы номенклатуры

Позволяет настроить отборы по папкам (родителям) номенклатуры, ценовым группам или категориям. Задать настройку выводить только те позиции, которые есть в наличии или номенклатуру без цен. Так же из данной закладки можно посмотреть те запасы, которые исключены из данного прайс-листа.

 

Виды цен (колонки с ценами)

Закладка содержит табличную часть, в которую добавляются колонки с ценами. Так же есть возможность ограничить вывод заголовков колонок с ценами.

 

Дополнительная информация

В данной закладке происходит настройка вывода контактной информации, подписи даты формирования, включение признаков новинок и представления остатков.

Следует отметить, что сам признак новинки хранится в номенклатуре, равно как и указания количественного признака границы для остатков. Еще данная закладка хранит текст подвала прайс-листа, который может содержать дополнительную текстовую информацию, например, краткое описание заключения сделок.

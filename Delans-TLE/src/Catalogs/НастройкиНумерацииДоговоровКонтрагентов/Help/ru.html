Справочник "Настройки нумерации договоров контрагентов"

Справочник предназначен для хранения настроек автоматической нумерации договоров контрагентов.

 

Справочник позволяет вести автоматическую нумерацию договоров контрагентов в трех вариантах:

1. В разрезе организации.

2. Организации + вид договора.

3. Организации + Группа договоров.

 

В случае, когда для договора указано больше одного варианта, программа определяет текущую настройку согласно приоритету (по убыванию):

Сначала организация + группа договоров, затем организация+ вид договора, и в последнюю очередь по организации.

 

Нумерация может быть периодичной в течение года, либо непериодической.

В случае, если необходимо, чтобы нумерация договора происходила по кнопке "Получить новый номер" в форме договора, а не автоматически при создании договора, необходимо установить настройку "Нумеровать" в состояние "Вручную". Данная настройка влияет и на нумерацию договоров по умолчанию при создании новых контрагентов.

 

Нумерация договоров будет происходит по правилам, описанным в шаблоне формата номера.

 

При создании договора заполняются Организация и Периодичность, Формат номера и Текущий номер.

 

При заполнении текущего номера следует учитывать, что следующий договор будет создан с номером Текущий номер + 1.
